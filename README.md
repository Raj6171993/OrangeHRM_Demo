OrangeHRM Demo – Playwright Automation

A lightweight Playwright automation suite for testing the OrangeHRM Demo application.
This project uses a clean Page Object Model (POM), reusable utilities, and structured test organization to keep the code simple, readable, and scalable.

Features : 

Playwright with JavaScript,
Page Object Model (POM),
Organized test structure,
Reusable utilities and fixtures.
Cross‑browser execution,
Screenshots & HTML reports.

Project Structure

tests/        → All test cases  
pages/        → Page Object Model classes   
utils/        → Test Data  

Running Tests

npm install

npx playwright test

npx playwright test --headed

npx playwright show-report
